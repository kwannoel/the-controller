# Video Recording

Video recording fits managed mode better than relay mode.

## Relay mode

Relay mode controls the user's real browser tab. That makes full-session recording more invasive and less predictable. Prefer screenshots unless the user explicitly wants recording from the live browser.

## Managed mode

Use managed mode when you need clean reproducible video capture for:

- demos
- bug reports
- test evidence
- repeated automation runs

If the task turns into repeatable capture instead of live browsing, switch from relay mode to managed mode before recording.
